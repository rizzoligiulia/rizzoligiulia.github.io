Lymphoma sub-types classification

Image classification task in which we have investigated several Deep Learning architectures, resulting in an efficient combination of a CNN and RNN. Both networks are used to exploit the spatial relations of smaller patches of the original images: first, the features are extracted in small neighbourhoods using the CNN, then the data sequences are analyzed by the RNN. All the patches are collectively considered in the prediction of the Lymphoma sub-type.
Language: Python

Polarization in Pro-Life and Pro-Choice Network on Twitter

Network analysis on the topic of abortion based on the use of different hashtags and words to reinforce opinion. The different connections between the tweets, their text and their hashtags underlined political and ideological orientations. The project was held in a team, with the collaboration of the ​ Linguistic and Literary Studies Department (DISLL) that defined the field of interest.
Language: Matlab and Python

SARS-CoV-2 gene analysis

Project following the COVID-19 outbreak: exploitation of genomics computational techniques to underline differential gene expression, gene ontology and discovering gene clusters. The comparison has been made considering a control group and running several statistical tests looking for significant dissimilarities.
Language: R

Latency-based approaches for TCP/IP networking

Latency is increasingly becoming a performance bottleneck for Internet Protocol (IP) networks, but historically, networks have been designed with aims of maximizing throughput and utilization. Delays can be introduced at various stages of communication. In TCP end-point interaction two aspects can be taken into account to accomplish latency reduction: improving the 3-way handshake when resuming a session and an early detection in packet loss recovery.

Frame extraction and Classification

A traditional computer vision approach is proposed to extract photo frames from sample images using C++ and OpenCV library. The extracted frames are then formatted and the predictions are computed on a Convolutional Neural Network which is pre-trained using CIFAR10 dataset.
Language: C++ and Python
